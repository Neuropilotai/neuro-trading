## Step 3: Paste into TradingView

1. Go to **[TradingView.com](https://www.tradingview.com)**
2. Open any chart (BTC/USD, SPY, etc.)
3. Click **"Pine Editor"** at the bottom of the screen
4. **Delete ALL** the default code
5. **Paste** your code (Cmd+V / Ctrl+V)
6. Click **"Save"** (top right)
7. Click **"Add to Chart"** (top right)
